#  Simulator Editor(模拟器编辑器)

这是一个基于 Web 的文字模拟游戏引擎。
This is a web-based text simulation game engine.

## 如何游玩 (How to Play)
- **在线玩 (Play Online)**: [点击这里 (Click Here)](https://your-username.github.io/my-villain-game/) *(请替换为您的 GitHub Pages 链接)*
- **本地玩 (Play Locally)**: 下载本项目，双击 `index.html` 即可启动编辑器，点击 "播放" 图标开始游戏。

## 功能特性 (Features)
- 📝 **内置编辑器**: 可视化编辑世界、天赋、事件和结局。
- 🌍 **多世界支持**: 支持修仙、都市等多种题材模板，同时可以自建世界观并设置特色属性词条。
- 🤖 **AI 辅助创作**: 提供提示词，一键生成完整剧本。
- 💾 **独立发布**: 一键导出单文件 HTML，方便分享。

## 开发 (Development)
- `index.html`: 游戏编辑器 (Game Editor)
- `game.html`: 游戏引擎核心 (Game Engine Core)
- `AI剧本创作提示词.md`: AI Prompt for content generation
